# HoYoCal

This repo contains ics files to be imported to any calendar application/program.

This repo is automatically updated by a python script that gets event information using [genshin.py](https://github.com/seriaati/genshin.py) for hoyogames

Endfield events are scraped from [game8](https://game8.co/games/Arknights-Endfield/archives/535443)
